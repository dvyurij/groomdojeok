사용방법

1. 브라우저에서 F12를 눌러 HTML elements를 본다.
2. ▶<body>부분 라인을 우클릭한다.
3. copy->copy outerHTML을 클릭하여 클립보드에 HTML을 복사한다.
4. 그 상태에서 groom프로그램 엔터키를 누른다.
5. 클립보드에 코드부분만 복사된다.
6. 입력할 곳에 붙여넣기 하고 적당히 손본다.
